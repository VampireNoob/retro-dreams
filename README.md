# 🚗 Retro-Dreams 

A responsive landing page for a fictional vintage car rental company, built with plain HTML, CSS, and vanilla JavaScript. Focused on smooth scroll animations (GSAP) and a warm, classic design that matches the retro theme.

**🔗 Live Demo:** [vampirenoob.github.io/retro-dreams](https://vampirenoob.github.io/retro-dreams/)

## Features

- Responsive design for desktop, tablet, and mobile (multiple breakpoints)
- Mobile hamburger navigation with animated toggle
- Scroll and load animations powered by [GSAP](https://gsap.com/)
- Second page with a clear pricing table
- Direct booking request via email link for each vehicle
- Contact section with email, WhatsApp, and Instagram

## Tech Stack

- HTML5
- CSS3 (Flexbox, Media Queries)
- Vanilla JavaScript
- [GSAP 3](https://gsap.com/) for animations
- Google Fonts (Della Respira)

## Project Structure

```
retro-dreams/
├── index.html          # Home page
├── preise.html         # Pricing page
├── styles.css          # All styling
├── script.js           # GSAP animations + navigation logic
├── favicon.ico
├── video.mp4            # Hero video
└── *.jpg                # Vehicle images
```

## Running Locally

```bash
git clone https://github.com/VampireNoob/retro-dreams.git
cd retro-dreams
```

Then simply open `index.html` in your browser — no build process or server required.
