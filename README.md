# 🚗 Retro-Dreams

Eine responsive Landingpage für eine fiktive Oldtimer-Vermietung, gebaut mit reinem HTML, CSS und Vanilla JavaScript. Fokus auf sanfte Scroll-Animationen (GSAP) und ein klassisches, warmes Design passend zum Retro-Thema.

**🔗 Live Demo:** [vampirenoob.github.io/retro-dreams](https://vampirenoob.github.io/retro-dreams/)

## Features

- Responsive Design für Desktop, Tablet und Mobile (mehrere Breakpoints)
- Scroll- und Load-Animationen mit [GSAP](https://gsap.com/)
- Zweite Unterseite mit übersichtlicher Preistabelle
- Direkte Buchungsanfrage per E-Mail-Link je Fahrzeug
- Kontaktbereich mit E-Mail, WhatsApp und Instagram

## Tech Stack

- HTML5
- CSS3 (Flexbox, Media Queries)
- Vanilla JavaScript
- [GSAP 3](https://gsap.com/) für Animationen
- Google Fonts (Della Respira)

## Struktur

```
retro-dreams/
├── index.html          # Startseite
├── preise.html         # Preistabelle
├── styles.css          # Gesamtes Styling
├── script.js           # GSAP-Animationen
├── favicon.ico
├── video.mp4            # Hero-Video
└── *.jpg                # Fahrzeugbilder
```
## Lokal ausführen

```bash
git clone https://github.com/VampireNoob/retro-dreams.git
cd retro-dreams
```

Danach einfach `index.html` im Browser öffnen – kein Build-Prozess oder Server nötig.

## Hinweis

Alle Kontaktdaten und die Firma "Retro-Dreams" sind fiktiv und dienen ausschließlich als Demo-Projekt.
